## 🤝 协作方式

本项目由 **阿原** 与 **OpenClaw AI Assistant** 协作完成。

### 如何协作

1. **通过 Feishu 发送指令**
   - `"给项目添加新功能"` → AI 自动开发并推送
   - `"更新 README"` → AI 编辑并提交
   - `"创建新页面"` → AI 编写代码并部署

2. **代码审查模式**
   - AI 提交到 feature 分支
   - 人类审核后合并到 main

3. **全自动模式**
   - AI 自主决策、开发、测试、部署
   - 完成后通知人类
