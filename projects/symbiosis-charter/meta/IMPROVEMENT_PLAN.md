## Quality Standards

### Rule Format Template
```markdown
**规则 [编号]【规则名称】**
[规则正文，使用a), b), c)分项]
a) [具体条款]
b) [具体条款]
c) [具体条款]

*[可选：补充说明或案例]*
```

### Content Requirements
- Each rule must be actionable and verifiable
- Rules must reference other rules where applicable
- Rules must include quantitative thresholds where appropriate
- Rules must specify responsible parties

### Consistency Checks
- [ ] Rule numbering follows hierarchical format
- [ ] All rules use consistent terminology
- [ ] Cross-references are accurate
- [ ] Chinese and English terms align
