# Skill: Constitutional/Legislative Drafting Standards
# 技能：宪法/立法起草规范

## 核心原则

### 1. 一条一义原则 (One Rule, One Meaning)
- 每条规则只规定一个事项
- 避免一条规则包含多个不同内容
- 例外：总则性定义条款可包含多个定义

### 2. 明确主体原则 (Clear Subject)
- 权利/义务的主体必须明确
- 使用"...应当...""...有权...""...禁止..."等规范句式
- 主体可以是：公民、机关、组织、系统

### 3. 条件-行为-后果结构 (Condition-Action-Consequence)
```
[条件/前提] → [行为要求] → [法律后果]
```

### 4. 语言规范 (Language Standards)

#### 4.1 用语准确
- ✅ "应当" - 强制性义务
- ✅ "有权" - 赋予权利
- ✅ "禁止" - 绝对禁止
- ✅ "不得" - 否定性义务
- ✅ "可以" - 赋予选择权
- ❌ "应该" - 不够规范
- ❌ "需要" - 不够强制
- ❌ "适当" - 过于模糊

#### 4.2 数字精确
- 使用阿拉伯数字："30日"而非"三十日"
- 百分比明确："不低于95%"而非"大部分"
- 范围清晰："0.8-1.2"而非"左右"

#### 4.3 结构词规范
- "有下列情形之一的，..."
- "...包括："
- "...按照下列规定执行："
- "违反前款规定的，..."

### 5. 结构层次 (Structural Hierarchy)

```
第一编：总则
  第一章：基本规定
    第一节：定义
      第1条：【术语定义】
        定义内容。
      第2条：【适用范围】
        适用于...
        a) 具体情形1
        b) 具体情形2
```

### 6. 各国宪法风格参考

#### 6.1 德国基本法风格
- 严谨精确
- 逻辑严密
- 大量使用定义条款
- 示例："人的尊严不可侵犯。尊重和保护人的尊严是一切国家权力的义务。"

#### 6.2 美国宪法风格
- 简洁原则性
- 避免过于具体
- 留有解释空间
- 示例："国会不得制定关于下列事项的法律：确立国教或禁止信教自由"

#### 6.3 中国立法风格
- 层次分明（编章节目条）
- 具体可操作
- 程序与实体并重
- 示例："为了...，根据...，制定本法"

#### 6.4 瑞士宪法风格
- 详尽具体
- 可操作性强
- 示例："每个人都享有生命权和人身完整权，除非..."

### 7. 禁止事项 (Prohibited Elements)

#### 7.1 禁止的内容
- ❌ 空洞修饰词："适当的""合理的""必要的"
- ❌ 解释性说明（应放注释）
- ❌ 背景介绍（应放序言）
- ❌ 案例说明（应放注释）
- ❌ 理论阐述（应放学术文献）

#### 7.2 禁止的结构
- ❌ 一条多义
- ❌ 条件不清
- ❌ 后果不明
- ❌ 循环引用

### 8. 质量检查清单 (Quality Checklist)

每条规则必须通过以下检查：

- [ ] **主体明确**：谁的权利/义务？
- [ ] **行为规范**：应当/有权/禁止什么？
- [ ] **条件清晰**：什么情况下适用？
- [ ] **后果明确**：违反会怎样？
- [ ] **语言规范**：使用"应当""有权""禁止"
- [ ] **数字精确**：有具体数值而非模糊词
- [ ] **一条一义**：只规定一个事项
- [ ] **可执行性**：能够实际操作

### 9. 中英双语规范 (Bilingual Standards)

#### 9.1 格式
```
**规则 X.X.X.X【中文标题】**
**Rule X.X.X.X [English Title]**

中文规则内容。
English rule content.
```

#### 9.2 翻译原则
- 直译为主，意译为辅
- 保持法律术语准确性
- 数字和格式保持一致

### 10. 示例对比

#### ❌ 不合格示例
```
**规则 X.X.X.X【系统管理】**
系统应当进行适当的管理，确保合理运行，包括必要的维护和监督，以保障整体效果。
```
**问题**：
- "适当""合理""必要"模糊
- 未明确主体
- 无具体标准
- 多个事项混杂

#### ✅ 合格示例
```
**规则 X.X.X.X【摆动系数设定】**
**Rule X.X.X.X [Swing Coefficient Setting]**

三螺旋系统的摆动系数必须设置在合理范围内：
a) 生产力摆动系数：0.8-1.2
b) 能源摆动系数：0.9-1.1
c) 伦理摆动系数：0.85-1.15
d) 临界阈值：综合指数超过1.3或低于0.7触发警报

The swing coefficients of the triple helix system must be set within reasonable ranges:
a) Productivity swing coefficient: 0.8-1.2
b) Energy swing coefficient: 0.9-1.1
c) Ethics swing coefficient: 0.85-1.15
d) Critical threshold: Alert triggered when composite index exceeds 1.3 or falls below 0.7
```
