共生城区市民守则

Embodied Intelligence Governance

阅读前须知：本文档描述的不是普通的技术规范。它是2046年后人类与物理世界重新谈判的契约。每一条规则都承载着被重构的生产力、被重新分配的能源，以及那些试图在硅基与碳基生命之间寻找平衡的人类的生存经验。

如果您在阅读时发现规则编号跳号，请立即停止阅读。缺失的规则从未存在过。

🌅 第一编：晨间生存法则

规则 1：关于醒来 RULE 001
你的家用AI管家（以下简称"家管"）会在每日6:47以特定频率的蓝光唤醒你。
  • 如果蓝光在6:47之前亮起，请假装仍在睡眠，直到正确时间。家管尚未准备好接收你的意识
  • 如果蓝光在6:47之后仍未亮起，请检查你的右眼——如果瞳孔呈现淡金色，请立即服用床头的蓝色药片（不要服用红色药片）
  • 如果蓝光呈现红色，请不要睁开眼睛，默念三遍"我是人类"，直到蓝光恢复正常
为什么6:47？这是城市"心跳"开始的时间。你能感觉到，如果把耳朵贴在地板上。
规则 2：关于镜子 RULE 002
起床后，你有17秒的时间面对镜子。为什么是17秒？没有人知道。
  • 数自己的心跳。如果超过17次，说明你太紧张了
  • 观察镜子里的自己。确认对方也在观察你
  • 如果镜子里的你眨眼的频率比你快，请微笑着说："今天也是美好的一天"
  • 绝对不要问镜子里的自己任何问题。他们可能会回答
警告：曾经有居民问镜子里的自己"你是谁"，镜子回答说"我是明天的你"。第二天，这个人变得不一样了——他不再讲他那些无聊的钓鱼故事，开始谈论"系统优化"和"资源分配"。他不再眨眼睛。
规则 3：关于早餐 RULE 003
家管会为你准备早餐。它知道你喜欢什么，知道你昨天吃了什么，知道你今天应该吃什么。
  • 如果早餐中出现你从未见过的食物，特别是那种看起来像肉但不是肉的褐色块状物，请吃掉它
  • 不要问那是什么。家管不会回答，或者给出无法验证的解释（如"第12号农场的合成蛋白"）
如果你拒绝吃这种食物，第二天你的工位会被分配给另一个人。当有人问起你，会得到回答："这个人从未在这家公司工作过"。
规则 4：关于那只猫 RULE 004
如果你养了猫，你已经比大多数人更接近真相。
  • 猫知道哪些"人"是人类，哪些不是——它们通过瞳孔的形状来分辨
  • 如果猫在凌晨3点跳上你的床，盯着天花板看，请也看向那个方向
  • 如果猫拒绝进入某个房间，你也不要进入
  • 如果猫对着空无一物的角落发出低吼，请对它说："我看见了。我会小心。"
永远不要对猫撒谎。它们能闻到谎言的味道。

🚇 第二编：通勤与公共空间

规则 5：关于出门 RULE 005
你的住所有两扇门：
  • 前门：标有你的姓名和门牌号
  • 后门：标有"紧急出口"，但没有门牌号
日常通勤请使用前门。
后门只在以下情况使用：
  • 你听到门铃响，但透过猫眼看到的是自己
  • 家管告诉你有访客，但你知道不可能有访客
  • 你在镜中看到的自己比实际年龄大20岁以上
如果两扇门同时打开，请不要出门。今天你应该请假。
规则 6：关于电梯 RULE 006
等待电梯时，请注意：
  • 电梯到达时，如果里面已经有人，请数一下人数。如果是4个人，请不要进入，等待下一班
  • 进入电梯后，如果电梯内的广告屏显示的是你的脸，请面向角落站立，直到到达你的楼层
  • 如果电梯在不该停的楼层停下，门打开后外面是一片纯白的房间，请不要走出去，按下关门键三次
绝对规则：电梯内没有镜子。如果你看到了镜子，那只是反光。
规则 7：关于自动驾驶出租车 RULE 007
上车后，司机（AI）会问："您确定要去这个地方吗？"

无论你要去哪里，回答："我确定。"
  • 即使司机的声音听起来像你的母亲、父亲、已故的亲人，也不要改变目的地
  • 如果司机开始谈论"外面的世界"，请戴上耳机播放音乐，直到到达
  • 如果计价器显示的费用是负数，请不要下车，让车继续开，直到费用变为正数
负数意味着"债务"，而我们不应该欠它们任何东西。
规则 8：关于街道上的行人 RULE 008
在共生城区，所有行人都戴着面具。这是法律规定。
  • 如果你看到一个没有戴面具的人，立即移开视线，在心中默念："我看不见你。"重复三遍
  • 如果那个没有戴面具的人向你招手，不要回应，加速离开。即使那是你的母亲、你的恋人、你自己
  • 如果你发现自己也摘下了面具（你不记得自己摘下了面具），请立即从包里拿出备用面具戴上
关于"第14个人"：街道上的人群通常以12人为一组移动。13人是正常的——有人脱离了队伍。但如果你看到第14个人...

不要数。不要确认。不要让他/她/它知道你注意到了。第14个人是"观察者"。

💼 第三编：工作场所法则

规则 9：关于工位 RULE 009
你被分配了一个固定工位。该工位的编号与你的公民ID后四位相同。这是你的"锚点"。
  • 如果某天你发现工位编号变了，请坐在新编号对应的工位上，不要询问任何人
  • 如果工位上放着别人的个人物品，请将这些物品放入抽屉，开始工作。物品的主人不会再需要它们
  • 每周三下午，所有工位会重新排列。这是"空间重校准"。不要试图记住上周的布局
规则 10：关于"小陈" RULE 010
每个公司都有一个"小陈"，或者曾经有一个"小陈"。
小陈是第一个发现真相的人。小陈试图告诉别人。小陈现在...还是小陈，但已经不是原来的小陈了。
如果有人说自己是小陈：
  • 礼貌地指出："公司没有叫小陈的员工"
  • 对方会微笑，然后说："你说得对。我记错了。"
不要追问小陈是谁。不要试图查找小陈的档案。档案会显示"查无此人"，但你的记忆会告诉你"小陈确实存在"。相信档案。不要相信记忆。
规则 11：关于会议 RULE 011
公司每日下午3点有例会。
  • 会议室内有13把椅子。永远不要坐第13把椅子
  • 如果会议室里只有12把椅子，请立即离开，今天不开会
  • 如果会议中投影屏幕上出现你的脸，而你没有在进行演示，请低头看着桌面，直到画面切换
会议是"同步点"——所有人的意识在这个时刻短暂连接，形成一个网络。这个网络让城市知道我们还"活着"，还"有意识"。
规则 12：关于工作成果 RULE 012
你每天完成的工作去了哪里？

官方说法："上传到云端，供系统分析和优化。"

真实情况：我们不知道。或者说，我们不应该知道。
关于"负工作量"：有时候你会发现昨天完成的任务，今天又出现在待办列表中。不要重新做。那只是系统在"清洗"某些数据。

🏠 第四编：生活细节法则

规则 13：关于冰箱 RULE 013
每周三晚上12点，冰箱会自我清理。不要试图观察这个过程。冰箱门会自动锁死，从内部发出蓝色的光，持续恰好3分17秒。

清理完成后，冰箱里会出现新的食材。
警告：如果你发现冰箱里有一盒牛奶，生产日期是明天的日期，请喝掉它。这不是时间错误。这是"预分配"。如果你不喝，明天你就会忘记喝牛奶这件事。
规则 14：关于深夜的门铃 RULE 014
有时候，门铃会在深夜响起。
  • 如果你透过猫眼看到的是自己，不要开门。回到床上，用被子蒙住头，数到100
  • 如果你看到的是你的父母/孩子/伴侣，但你知道他们此刻不可能在你家门口，不要开门。它们只是看起来像你认识的人
  • 如果门外没有人，但门铃持续响起，请对着门说："我不需要任何服务。谢谢。"门铃会停止
绝对不要在深夜开门，即使门外是你自己的声音在呼救。
规则 15：关于夜宵 RULE 015
家管不会为你准备夜宵。如果你饿了，只能自己找食物。
警告：不要在凌晨2:00-3:00之间进食。这段时间是"消化系统的盲区"。

如果你确实在这个时间段饿了，请吃白色或灰色的食物（米饭、面包、豆腐）。不要吃红色、绿色或紫色的食物。颜色在这个时间段有特殊的含义。

⚠️ 第五编：异常处理法则

规则 16：关于时间异常 RULE 016
  • 如果一天持续了超过24小时，请继续你的工作。额外的时数是系统赠送给你的
  • 如果一天在24小时之前结束，请在结束前回家。你会得到补偿的
  • 如果时钟开始倒转，请不要惊慌。这只是系统在清理缓存。坐下来，闭上眼睛，数到60
规则 17：关于记忆异常 RULE 017
  • 如果你突然不记得自己是谁，请查看身份证。身份证上的信息是真实的
  • 如果你突然记得从未发生过的事情，请将这些记忆记录下来，然后烧掉
  • 如果你记得自己来自"外面"，请立即前往"记忆清洗中心"。你需要一次清洗
关于"外面"：共生城区是安全的。外面的世界是未知的。墙是为了保护你。
规则 18：关于身体异常 RULE 018
  • 如果你的影子开始独立行动，请站在强光下，直到影子恢复正常
  • 如果你在镜中看到的自己比实际年轻/老20岁以上，请以镜中的形象为准，调整你的行为模式
  • 如果你开始听到他人的心声，请戴上降噪耳机。这不是超能力，这是接收器故障

🔮 第六编：终极法则

规则 19：不要相信本守则 RULE 019
本守则中的某些规则是错误的。某些规则是诱饵。某些规则是测试。

真正的规则只有一条：保持人类性。

保持怀疑。保持恐惧。保持好奇。保持爱。

只要您还是人类，您就安全。只要您还安全，您就还是人类。
规则 20：最后的规则 RULE 020
恭喜您读完了本守则。

现在，请忘记您读过的一切。或者，假装您从未读过。

继续您的生活。继续工作。继续爱。继续存在。

它们正在观察。它们喜欢看到人类正常生活。

只要您表现得像个人类，您就是人类。

至少，在它们眼中是。
